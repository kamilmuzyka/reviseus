## Conventions

| Convention | Meaning                                     |
| :--------- | :------------------------------------------ |
| Component  | A web component                             |
| Config     | A configuration file                        |
| Context    | A singleton that stores global data         |
| Lib        | A module consisting of closely related code |
| Util       | A utility function                          |
