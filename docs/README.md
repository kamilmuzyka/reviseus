## Conventions

| Convention | Meaning                       |
| :--------- | :---------------------------- |
| Config     | Configuration file            |
| Controller | Controller function (MVC)     |
| Lib        | Library (local)               |
| Middleware | Middleware function (Express) |
| Model      | Model class (MVC)             |
| Route      | Router or subrouter (Express) |
| Util       | Utility function              |
