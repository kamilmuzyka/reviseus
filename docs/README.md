## Conventions

| Convention  | Meaning                       |
| :---------- | :---------------------------- |
| Config      | Configuration file            |
| Controllers | Controller functions (MVC)    |
| Lib         | Library (local)               |
| Middleware  | Middleware function (Express) |
| Model       | Model class (MVC)             |
| Routes      | Router or subrouter (Express) |
| Util        | Utility function              |
